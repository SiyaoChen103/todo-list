# To Do List 

一个简洁温馨的待办事项管理应用，支持拖拽排序、截止日期、置顶、日历视图等功能。

## 功能特性

### 核心功能
- ✅ 添加待办事项，支持设置截止日期和时间
- ✅ 鼠标拖拽调整待办事项优先级顺序
- ✅ 标记完成/归档/删除待办事项
- ✅ 置顶重要待办事项，置顶项可单独排序

### 统计模块
- ✅ 今日待办数量
- ✅ 今日已完成数量  
- ✅ 今日到期数量
- ✅ 点击统计卡片筛选对应事项

### 日历视图
- ✅ 右侧显示当月日历
- ✅ 有待办事项的日期有标记提示
- ✅ 点击日期查看当天所有待办事项
- ✅ 可在日历中直接完成/取消完成

### 其他
- ✅ 删除确认弹框，防止误触
- ✅ 编辑待办事项内容和日期时间
- ✅ 已归档事项可恢复或彻底删除
- ✅ 数据本地持久化（localStorage）
- ✅ 响应式布局，支持桌面端和移动端
- ✅ 温馨暖色调低饱和配色

## 启动方式

### 直接打开（最简单）
```bash
# 找到项目目录，双击 index.html 文件
# 或者用浏览器打开
open ~/todolist/index.html
```

### 本地服务器（推荐）
如果你想要更好的体验，可以使用简单的 HTTP 服务器：

```bash
# 进入项目目录
cd ~/todolist

# 使用 Python 启动服务器
python3 -m http.server 8080

# 或者使用 Node.js (如果已安装)
npx serve .
```

然后在浏览器访问：`http://localhost:8080`

## 技术栈

- 纯 HTML + CSS + JavaScript（Vanilla）
- 单文件设计，所有代码在一个 HTML 文件中
- localStorage 数据持久化

## 项目结构

```
todolist/
└── index.html    # 整个应用（单文件）
```

## 使用说明

1. 在输入框中输入待办事项内容
2. 可选择设置截止日期和时间
3. 点击"添加"或按回车键添加
4. 拖拽事项调整顺序
5. 点击 📌 置顶事项
6. 点击 ✓ 标记完成
7. 点击日历日期查看当天事项
